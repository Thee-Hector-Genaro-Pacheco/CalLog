# CalLog

Calibration & instrument asset tracker with due-date visibility, calibration records, and certificate-style reporting.

## Tech Stack
- TypeScript
- Node.js + Express
- GraphQL
- MongoDB (Atlas)
- Next.js (web UI)
- Vercel (web hosting)

## Apps
- `apps/api` — GraphQL API
- `apps/web` — Next.js UI (coming next)

## Quick Start (API)
```bash
npm install
cp apps/api/.env.example apps/api/.env
npm run dev:api
